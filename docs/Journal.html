Adding R stuff

So far this is just a blog where you can write in plain text and serve your writing to a webpage. One of the main purposes of this lab journal is to record your progress learning R. The reason I am asking you to use this process is because you can both make a website, and a lab journal, and learn R all in R-studio. This makes everything really convenient and in the sam place.

So, let’s say you are learning how to make a histogram in R. For example, maybe you want to sample 100 numbers from a normal distribution with mean = 0, and standard deviation =1, and then you want to plot a histogram. You can do this right here by using an r code block, like this:

samples <- rnorm(100, mean=0, sd=1)
hist(samples)

When you knit this R Markdown document, you will see that the histogram is printed to the page, along with the R code. This document can be set up to hide the R code in the webpage, just delete the comment (hashtag), from the cold folding option in the yaml header up top. For purposes of letting yourself see the code, and me see the code, best to keep it the way that it is. You learn all of these things and more can be customized in each R code block.

# Solution #1

sumFunction = function(firstInteger, secondInteger) {
  
  # this function finds the sum of all the integers between two integers but not including the bounded integers
  
  # first identifies the larger integer and smaller integer
  biggerInteger = max(firstInteger, secondInteger)
  smallerInteger = min(firstInteger, secondInteger)
  result = 0
  
  # adds to the result variable given the condition that the two integers are at least one integer apart
  while((biggerInteger - smallerInteger) > 1 ) {
    smallerInteger = smallerInteger + 1
    result = result + smallerInteger }
  
  return(result)
}

sumFunction(1,100)
## [1] 4949

# The goal is to identify the composite numbers first
listOfNumbers = c(1:1000)
compositeIndex = c()

for(n in 1:length(listOfNumbers)) {
  for(d in 1:length(listOfNumbers)) {
    # the if statement determines if a number in the list is composite
    if((listOfNumbers[n]/listOfNumbers[d] > 1 && listOfNumbers[n] %% listOfNumbers[d] == 0 && listOfNumbers[d] != 1) || listOfNumbers[n] == 1) {
      compositeIndex = append(compositeIndex, n) }
  } }

compositeIndex = unique(compositeIndex)
# eliminates the composite numbers from the list
listOfNumbers = listOfNumbers[-compositeIndex]
